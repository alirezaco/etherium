# start with npm for dev

### 1) cd nest-etherium

### 2) npm install

### 3) npm run start:dev

<br>
<br>

# start with npm for product

### 1) cd nest-etherium

### 2) npm install

### 3) npm run build

### 3) npm run start:prod

<br>
<br>

# start with docker for product

### 1) cd nest-etherium

### 2) docker image build -t ETH:1 .

### 3) docker run -p 3000:3000 ETH:1
